# PHASE 5 - CART MODULE (FRONTEND + BACKEND)

Read **AGENT.md** first and follow it STRICTLY.

Follow **PHASE 0 - GLOBAL INSTRUCTIONS** before implementing anything.

Use the Authentication module from Phase 2.

Use the Product module from Phase 3 and Phase 4.

This phase is ONLY for Cart functionality.

Do NOT implement:

* Checkout
* Orders
* Payment Gateway
* Wishlist
* Coupons
* Shipping
* Order History
* Real-time cart updates
* Socket.io

Implement Cart only.

---

## Objective

Build a complete Cart Module where authenticated users can:

* Add products to cart
* View cart
* Increase quantity
* Decrease quantity
* Remove items from cart
* See total price
* See total quantity

The cart should persist in MongoDB.

---

# Authentication Requirement

Cart is a protected feature.

Only:

```txt
Authenticated Users
```

can:

* Add to cart
* View cart
* Remove from cart
* Update quantity

Unauthenticated users:

Redirect to:

```txt
/ login
```

or return:

```txt
401 Unauthorized
```

---

# Cart Database Schema

Create:

```txt
models/Cart.ts
```

Schema:

```ts
{
user:ObjectId,

items:[

{

product:ObjectId,

quantity:number

}

]
}
```

Requirements:

* user references User collection
* product references Product collection
* quantity minimum 1
* timestamps enabled

# GET CART

Endpoint:

```txt
GET /api/cart
```

Requirements:

Return:

```ts
{

success:true,

data:{

items:[

{

product,

quantity

}

],

totalItems,

totalPrice

}

}
```

Populate:

```txt
product
```

Return:

```txt
title

price

discount

image

stock
```

---

# ADD TO CART

Endpoint:

```txt
POST /api/cart
```

Request:

```json
{

productId:"",

quantity:1

}
```

Requirements:

If product already exists:

```txt
Increase quantity
```

Else:

```txt
Add new item
```

Before adding:

Check:

```txt
Product exists

Stock available
```

Return:

```json
{

success:true,

message:"Product added to cart"

}
```

---

# UPDATE CART QUANTITY

Endpoint:

```txt
PUT /api/cart
```

Request:

```json
{

productId:"",

quantity:2

}
```

Requirements:

Update quantity.

If:

```txt
quantity <= 0
```

Remove item automatically.

Validate:

```txt
Available stock
```

---

# REMOVE CART ITEM

Endpoint:

```txt
DELETE /api/cart/[productId]
```

Requirements:

Remove item from cart.

Return:

```json
{

success:true,

message:"Item removed from cart"

}
```

# Price Calculation

Calculate:

```txt

Subtotal

```

```txt
Final Price

=

price

-

(price × discount /100)
```

