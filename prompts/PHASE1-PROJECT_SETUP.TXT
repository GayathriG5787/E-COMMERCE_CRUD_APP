# PHASE 1 - PROJECT SETUP

Read **AGENT.md** first and follow it STRICTLY.

Follow **PHASE 0 - GLOBAL INSTRUCTIONS** before implementing anything.

## Objective

Set up the foundation of the Full Stack E-commerce Clothes application using the technology stack defined in AGENT.md.

This phase is ONLY for project setup.

Do NOT implement:

* Authentication
* Product CRUD
* Cart
* Admin Dashboard
* Middleware
* API business logic

Only prepare the project structure and configuration.

---

## Technology Stack

Use ONLY:

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS v4
* Shadcn UI
* Redux Toolkit
* MongoDB Atlas
* Mongoose
* Axios

Use stable versions compatible with:

* Next.js 15
* React 19
* Tailwind CSS v4

Follow AGENT.md version stability rules.

---

## Global CSS

Create:

```txt
src/app/globals.css
```

Define:

```css
:root {

--background:

--foreground:

--primary:

--secondary:

--muted:

--success:

--destructive:

--border:

--card:

}
```

Use the colors defined in AGENT.md.

Create reusable classes:

```css
.btn-primary

.page-container

.card-container

.input-field

.section-title
```

Avoid repeating long Tailwind utility combinations throughout the project.

---

## MongoDB Configuration

Create:

```txt
lib/mongodb.ts
```

Requirements:

* Single connection pattern
* Reuse existing connection
* Avoid reconnecting on every request
* Use Mongoose
* Read MongoDB URI from:

```txt
.env.local
```

Do NOT:

* Hardcode database URLs
* Create multiple connections

---

## Environment Variables

Create:

```txt
.env.local
```

Include:

```env
MONGODB_URI=

JWT_SECRET=

NEXT_PUBLIC_API_URL=
```

Never hardcode:

* Secrets
* API Keys
* Database URLs
* Tokens
